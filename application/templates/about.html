About XRD.Qanalyze

XRD.Qanalyze is an automated cloud-based application for quantitative analysis of mineral samples using X-ray diffraction (XRD) data. It is currently free of charge and requires a google/gmail login.

Qanalyze applies full pattern fitting methods to compute mineral abundances from X-ray diffraction data.
Qanalyze brings unsurpassed user-friendliness to XRD data analysis.

While Qanalyze principles are not limited to a particular type of XRD instrument, it is currently focused on data collected with portable and benchtop XRD instruments.

Qanalyze currently works best with a limited selection of phase candidates.
See and change the list in Settings > Set Phases.

Instrumental parameters vary from system to system.
For best results, set your instrument parameters in Settings > Calibration Parameters.

Instructions

1. Upload your XRD pattern (formats currently accepted: txt, plv, mdi, dif)

2. Wait a few seconds

3. Review the results in a graph overlaying experimental and calculated data and a table listing minerals found and abundances.

4. Download results in a CSV file.

About us

Qanalyze is developed by a dedicated team of experienced software engineers and instrument scientists based in California, USA.
